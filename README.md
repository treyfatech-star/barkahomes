# Royal Homes & Consults ltd Frontend

This app is a frontend-only React project for the Royal Homes & Consults ltd marketing site.

## Available Scripts

In the `frontend` directory, you can run:

### `npm start`

Starts the development server.

### `npm run build`

Builds the production bundle in `build`.

### `npm test`

Runs the test command through CRACO.

## Contact And Newsletter

- The contact form opens the visitor's default email app with a prefilled enquiry.
- The newsletter form opens the visitor's default email app with a prefilled signup request.
